# Interview Prep

## Core Concepts

- Threat modeling and risk assessment
- CI/CD security gates and exception handling
- SBOMs, signing, provenance
- Runtime detection and response

## Practical Questions

- How do you gate PRs without blocking delivery?
- How do you handle a critical vulnerability in production?
- How do you prove artifact integrity?
- How do you measure DevSecOps success?

## Exercises

- Walk through a lab and explain trade-offs
- Design a minimal pipeline for a new service
- Draft an incident response plan
