# Phase 06: Runtime Detection

## Goals

- Detect suspicious behavior in production
- Route alerts to responders with context

## Practices

- Runtime detection rules
- Centralized logging and metrics
- Alert routing and triage workflow

## Tools

- Runtime detection: [Falco](https://falco.org/)
- K8s security: [Trivy Operator](https://github.com/aquasecurity/trivy-operator)
- Observability: [OpenTelemetry](https://opentelemetry.io/), [Prometheus](https://prometheus.io/), [Grafana](https://grafana.com/)

## Deliverables

- Runtime detections and dashboards
- Alert routing to incident system
- Tuning backlog for noisy alerts
