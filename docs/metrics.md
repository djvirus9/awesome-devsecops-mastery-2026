# Metrics and KPIs

Measure security outcomes and operational performance.

## Core Metrics

- MTTA/MTTR for security findings
- Vulnerability SLA compliance rate
- SBOM coverage per release
- Signed artifact coverage
- CI gate pass rate
- Mean time between policy exceptions

## Reporting Cadence

- Weekly: CI failures, SLA breaches
- Monthly: SBOM coverage, top recurring issues
- Quarterly: maturity improvements and tool ROI
