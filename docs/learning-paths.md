# Learning Paths

Focused paths for different experience levels and roles.

## Beginner (2-4 weeks)

1. Complete labs 01-03
2. Set up CI PR gates and branch protection
3. Generate an SBOM and publish it

## Intermediate (4-8 weeks)

1. Add IaC scanning and policy-as-code
2. Add DAST to staging
3. Add runtime detection with alert routing

## Advanced (8+ weeks)

1. Add provenance attestations
2. Implement risk-based policy enforcement
3. Build detection engineering and IR playbooks
